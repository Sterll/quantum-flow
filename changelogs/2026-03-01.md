# Changelog - Quantum Flow Lib
## 2026-03-01

### ✨ Nouveautés / Améliorations
- **Architecture modulaire complète** : Remplacement de GraphModel/ConnectionValidator/History par un modèle v2 entièrement événementiel
- **EventBus type-safe** : Système d'événements avec on/off/emit/once pour la communication réactive
- **Validator composable** : 6 règles de validation intégrées pour assurer la cohérence du graphe
- **GraphStore** : Entrepôt de données avec événements, validation, transactions batch et helpers de requête
- **HistoryManager** : Gestion complète de l'historique avec auto-subscribe, transactions batch et labels
- **API defineNode améliorée** : Support UUID natif, validation des pins, catégorisation des nœuds
- **NodeRegistry enrichi** : Nouvelles méthodes unregister, has, getCategories
- **Rendus multiples** :
  - Renderer style UE5 Blueprint (nœuds fluides, connexions typées)
  - Renderer Obsidian (moteur, particules animées, nœuds multi-couches)
  - Renderer premium Canvas (pins, connexions Bezier, effet glow)
  - Renderer Claude Terminal (style professionnel et épuré)
- **FlowCanvas React** : Composant React avec rendu canvas intégré
- **Storybook** : Stories pour FlowCanvas et defineNode avec exemples concrets
- **Undo/Redo** : Système d'historique natif

### 🐛 Corrections
- Alignement react-dom v18 pour correspondre à la peer dependency de React
